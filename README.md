This repository contains some of the raw data and code files used to generate figures and perform analysis for my thesis on thsz1 - I've included a short description of each included file in the following README. 

ImmunoVolumeData: Raw volume data sourced from confocal imaging of 3.5 and 4.5 dpf embryos immunostained for zpr1
tshz1Alignment.ipynb: Jupyter lab python file containing code used to create sequence alignment figures
tshz1_all_sequences.fasta: fasta file containing unannotated DNA sequences of tshz1 for the WT, fh247, and CRISPANT 14 Male alleles
